# BitUrls

A url shortening web app made using MERN stack.

## Tech Stacks
- Frontend: Vite + React + TS
- Backend: Express + TS
- Database: MongoDB
- UI: Tailwindcss + Shadcn

## Steps to setup the environment
- Install the dependencies
  ```bash
    yarn install
  ```
  
- Start the development server
  ```bash
    yarn dev
  ```
